# AHB-Based Multi-Master System with Cache and APB Bridge

## Overview
Designed a simplified AMBA AHB multi-master bus architecture using Verilog RTL.

## Features
- Two AHB Masters
- Round Robin Arbiter
- 2-Stage Pipeline
- Address Decoder
- Cache Controller
- RAM Slave
- AHB to APB Bridge
- APB Peripheral

## Tools Used
- Verilog HDL
- Cadence Xcelium
- RTL Synthesis Tools

## Files
- src/design.v → RTL design
- tb/testbench.v → testbench
- docs/APP_CIAP_40.pdf → complete report
